# 🛒 쿠팡 카탈로그 데이터 정제 · 패턴 탐지 프로젝트
**Coupang Catalog Pattern Detection & Data Quality Check**

본 프로젝트는 쿠팡과 같은 이커머스 플랫폼의 **카탈로그 품질 관리(Data Quality Management)** 업무를 대비하기 위해  
공공데이터(가공식품 소매가격 정보)를 기반으로 **상품명 패턴 분석, 이상 패턴 탐지, 가격 이상치 검출**을 수행한 프로젝트입니다.

> ⚡ 실무에서 실제로 사용하는 품질 검증 로직을 그대로 구현하여  
> 쿠팡의 데이터 분석/카탈로그 정제 직무 포트폴리오로 활용할 수 있도록 구성했습니다.

---

## 🚀 프로젝트 개요

### 1) 사용 데이터
- 한국농수산식품유통공사 가공식품 소매가격 정보(2023~2025)
- 품목, 품종(상품명), 가격(평균/최저/최고), 조사일 등 포함
- 상품명 구조가 **브랜드 + 제품명 + 규격(용량)** 형태로 구성되어  
  **쿠팡 카탈로그 구조와 매우 유사**

### 2) 해결하고자 하는 문제
- 상품명 규칙 불일치 (단위 누락, 브랜드·용량 혼재)
- 단위/규격 오류
- 가격 데이터 이상치
- 비정상 패턴 탐지

---

## 🧠 프로젝트 주요 결과

### ✔ 텍스트 기반 상품명 패턴 분석 (TF-IDF + KMeans)
- 상품명을 벡터화하여 **패턴 클러스터 5개** 생성
- 정상 패턴군(브랜드+상품+규격) vs 비정상 패턴군 자동 분류

### ✔ 카탈로그 이상 패턴 탐지 (Isolation Forest)
- 규격 정보 누락  
- 숫자/단위 부재  
- 비정상적으로 짧은 상품명  
- 중복 단어 포함  
을 자동 탐지

### ✔ 가격 이상치 탐지 (Z-score)
- 조사일별 가격 시계열 생성  
- |z| > 3 기준으로 가격 오류/이상치 탐지  
- 비정상 고가/저가 날짜 검출
