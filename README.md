# 🛒 Coupang Data Analysis Portfolio

이 저장소는 커머스 환경에서 **상품 데이터 품질 개선**,  
특히 **카탈로그 라벨 오류 탐지(NLP 기반)**에 집중한 프로젝트 포트폴리오입니다.

쿠팡 직무에 맞게 재구성하여 정리했습니다.

---

## 📂 Projects Included

### 1️⃣ Catalog Label Error Detection (상품 라벨 오류 탐지)
상품명–카테고리–옵션 간 불일치 여부를 판별하는 모델 구축  
- NLP 기반 텍스트 전처리  
- TF-IDF / Word2Vec 특징 추출  
- 로지스틱 회귀 / 랜덤포레스트 비교  
- Precision–Recall 기반 검출 성능 개선  
👉 **폴더:** `01_catalog_label_error_detection/`

---

### 2️⃣ Review Scraping & Sentiment Analysis (리뷰 수집·감성분석)
웹스크래핑을 통한 리뷰 수집 → 감성분석 기반 인사이트 도출  
- 크롤링(Web scraping / API)  
- 감성사전 기반 분석  
- 토픽 모델링(LDA)  
👉 **폴더:** `02_review_sentiment_scraping/`

---

### 3️⃣ Text Mining Summary Projects (텍스트마이닝 종합)
대학원 텍스트마이닝 과제를 기반으로 한 분석 모음  
- 형태소 분석 비교(Mecab vs Okt)  
- 키워드 네트워크  
- 토픽 모델링 (LDA / NMF)  
👉 **폴더:** `03_text_mining_summary/`
